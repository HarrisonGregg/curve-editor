# curve-editor
## Requirements
c++11 and freeglut

##Controls
Draw curves by clicking while holding down one of the following keys:

 - p: polylines

 - b: Bezier Curves

 - r: Catmull-Rom Splines

 - l: Lagrange curves

Click curves to select.

Drag curves to move.

Hold 'a' to add points to selected curve.
